# 芯片型号命名规则

基于`ARM Cortex-M3`内核的增强型`32位MCU`

## STM32

* STM32：表示 ST 公司生产的 32 位 MCU 系列。

* F：产品类型，F 代表通用型 Flash MCU（2.0~3.6V 供电）。

* 103：子系列编号，103 为增强型，带 USB 接口及更多外设资源。

* C：引脚数量，C 表示 48 引脚（T=36，R=64，V=100，Z=144）。

* 8：Flash 容量，8 表示 64KB（4=16KB，6=32KB，B=128KB，C=256KB，E=512KB 等）。

* T：封装类型，T 为 LQFP（H=BGA，U=VFQFPN，Y=WLCSP）。

* 6：温度范围，6 表示 -40℃~85℃ 工业级（7 表示 -40℃~105℃）。

### 容量分类：

* 小容量：16~32KB（x4、x6）

* 中容量：64~128KB（x8、xB）

* 大容量：256~512KB（xC、xD、xE）

* 超大容量：512KB 以上（xF、xG）

### 常见型号示例：

* STM32F103C6T6：48 引脚，32KB Flash，10KB SRAM

* STM32F103C8T6：48 引脚，64KB Flash，20KB SRAM

* STM32F103RCT6：64 引脚，256KB Flash，48KB SRAM

* STM32F103VET6：100 引脚，512KB Flash，64KB SRAM

### STM32常见产品类型全览

| 子系列代号 | 内核 | 核心定位 / 主打特点 | 典型应用场景 |
| :--- | :--- | :--- | :--- |
| **主流级 (Mainstream)** | | | |
| `C0` | Cortex-M0+ | 入门级、超高性价比。旨在以接近8位/16位MCU的成本提供32位的性能，简化硬件设计，封装小巧。 | 烟感报警器、小家电、工业风扇等成本敏感型应用。 |
| `F0` | Cortex-M0 | 经济型、易用。作为8位/16位MCU的升级路径，性能与成本之间取得了良好平衡。 | 电池应用、物联网传感器、简单控制设备。 |
| `F1` | Cortex-M3 | 经典通用系列。市场应用最广、资料最丰富的系列之一。其中的103增强型是代表。 | 工业控制、电机驱动、消费电子、各种通用嵌入式系统。 |
| `G0` | Cortex-M0+ | 高效能、低成本。相比F0系列具有更高的集成度和能效，是F0的升级版。 | 智能锁、照明控制、电源管理、电机控制。 |
| `G4` | Cortex-M4 | 混合信号、高精度控制。增强的外设，尤其是高精度模拟外设（如多个数模转换器、运算放大器），适合实时控制。 | 数字电源、电机控制、高级工业控制系统。 |
| `H5` | Cortex-M33 | 高性价比、可扩展安全。主频250MHz，性能强劲，同时提供可配置的安全特性以应对不同安全等级需求。 | 物联网网关、工业自动化、需要安全功能的中端应用。 |
| `H7` | Cortex-M7 | 极致性能、图形处理。目前性能最强的STM32系列之一，主频高达600MHz，具备强大的计算和图形处理能力，支持边缘人工智能。 | 复杂人机交互界面、高级音频处理、边缘计算、人工智能。 |
| **超低功耗 (Ultra-low power)** | | | |
| `L0` | Cortex-M0+ | 超低功耗入门级。极致的静态功耗和快速唤醒能力。 | 电池供电设备、能量采集、传感器节点。 |
| `L1` | Cortex-M3 | 经典超低功耗。在低功耗和性能之间取得良好平衡，通常集成了LCD驱动器。 | 智能电表、医疗手持设备、GPS数据记录器。 |
| `L4` / `L4+` | Cortex-M4 | 高性能超低功耗。平衡DSP性能和低功耗，是目前市场主力，具有极佳的能效比。 | 可穿戴设备、物联网设备、活动追踪器、智能家居。 |
| `L5` | Cortex-M33 | 安全型超低功耗。在L4基础上引入了TrustZone硬件安全隔离技术，为物联网安全设计。 | 物联网安全设备、移动支付终端、身份验证产品。 |
| `U3` / `U5` | Cortex-M33 | 新一代标杆级超低功耗安全。采用更先进制程，能效比达到新高度，集成了更强的安全特性。 | 高端智能手表、医疗传感器、工业传感器、电池供电的高性能设备。 |
| **无线型 (Wireless)** | | | |
| `WB` 系列 | M0+ & M4 | 双核无线SoC。一个内核处理协议栈，另一个内核处理用户应用。 | 蓝牙低功耗应用、智能家居设备、信标。 |
| `WBA` 系列 | Cortex-M33 | 新一代单核无线SoC。支持低功耗蓝牙5.3标准，在单核上结合了高性能与安全性。 | 高性能蓝牙低功耗应用、音频传输、工业无线传感器。 |
